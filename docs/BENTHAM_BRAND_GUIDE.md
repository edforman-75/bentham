# Bentham Research Brand Guide

## Company Name
**Bentham Research**

## Tagline
AI Visibility Intelligence

## Report Footer Requirements

All Bentham Research reports must include the following copyright footer:

```
© 2026 Bentham Research. All Rights Reserved. Confidential.
This report and its contents are proprietary to Bentham Research.
Unauthorized reproduction, distribution, or disclosure is prohibited.
```

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Navy | #003366 | Primary brand color, headings, data table headers |
| Gold | #C9A227 | Accent color, insight box highlights, cover logo |
| Charcoal | #2C2C2C | Secondary text, body copy headings |
| Light Gray | #F5F5F5 | Background fills, alternate table rows |
| Medium Gray | #E0E0E0 | Borders, dividers |
| Accent Blue | #0066CC | Positive highlights, links |
| Accent Red | #CC3333 | Negative indicators, critical gaps |
| Accent Green | #228B22 | Positive indicators, success states |

## Typography

- **Headings:** Playfair Display (serif)
- **Body:** Source Sans Pro (sans-serif)
- **Fallback:** -apple-system, BlinkMacSystemFont, sans-serif

## Report Elements

### Cover Page
- Navy gradient background (#003366 to #001a33)
- Gold logo text with 4px letter spacing
- White heading text
- Gold horizontal rule (80px width, 2px height)

### Insight Boxes
- Navy background
- Gold left border (4px)
- White text
- Gold subheading

### Data Tables
- Navy header row
- Alternating white/light gray rows
- 1px medium gray borders

### Callouts
- Light gray background
- Navy left border (4px)
- Warning variant: Yellow background (#FFF8E6), gold border
- Critical variant: Pink background (#FFE6E6), red border

## Report Types

### VP Strategy Reports
Full-length strategic assessments including:
- Cover page
- Executive summary with key stats
- Surface-by-surface analysis
- Brand-specific deep dives
- Strategic recommendations
- Methodology appendix

### Quick Reports
Abbreviated reports for specific analyses (brand stores, content comparison, etc.)

## Confidentiality

All Bentham Research reports are confidential by default. Include "Confidential" on cover page and in footer.
